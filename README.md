# Project Title 
This program is a quick game where you control a square, and must touch the target squares to gain points     

## Description  
This is a user-friendly, operatable program run on Python's Pygame. It is an introductory practce to creating a full fledged game and can be played for an indefinite amount of time.     

## How To Run   
Simply press the run button on python, and the pygame window should pop up automatically. The controls consist of the four arrows keys on the keyboard, each of them matching the direction.   

##Features  
This game involves controlling the user (blue) square, and using arrow keys to make contact with the target (green) squares. The score goes up once there is a collision, and the target square should move to a random position on the window. There is no timer, so you may rack up as many points as you'd like.        

## Technologies used    
This program is made with Python, along with the pygame module.
